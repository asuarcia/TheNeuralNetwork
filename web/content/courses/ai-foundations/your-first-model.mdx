Forget neural networks for a moment. The simplest useful model is a **threshold rule**: if a number crosses a line, predict one thing; otherwise, predict the other. That's a real classifier, and spam filters started exactly here.

## The idea

Suppose something scores each email for "spamminess" from 0 to 1. We pick a **threshold** — say 0.5 — and decide:

```python
def predict(score, threshold=0.5):
    return "spam" if score > threshold else "not spam"

print(predict(0.9))   # spam
print(predict(0.1))   # not spam
```

That's it. The only "intelligence" is the threshold — and choosing a good threshold from data is what *learning* will do for us in later courses.

<Callout type="tip">
Every model, from this rule to GPT, is the same shape: <strong>input → some math → prediction</strong>. The models get fancier; the shape never changes.
</Callout>
