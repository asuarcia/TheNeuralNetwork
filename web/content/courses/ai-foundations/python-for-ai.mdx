Every AI tool you'll build is, underneath, a Python program shuffling numbers around. You only need a handful of building blocks to follow along — and you'll write them right here.

## Variables hold things

A **variable** is a labeled box. You put a value in, you take it out by name.

```python
learning_rate = 0.01
model_name = "tiny-net"
```

## Lists hold many things

A **list** is an ordered collection — exactly how we'll store datasets, predictions, and weights.

```python
scores = [88, 92, 79, 95]
print(scores[0])   # 88  (counting starts at zero)
```

<Callout type="tip">
Almost all data in machine learning is "a list of numbers" (or a list of lists). Get comfortable here and the rest feels familiar.
</Callout>

## Functions package up work

A **function** takes inputs, does something, and gives back a result. A model is really just a (very fancy) function.

```python
def double(x):
    return x * 2

print(double(21))   # 42
```

## Your turn

A **list comprehension** transforms every item in a list — you'll use these constantly to apply a model to a batch of inputs.
