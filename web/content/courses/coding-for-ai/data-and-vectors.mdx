A model can't read text or see a picture. It only does arithmetic. So the first step in every AI system is turning real things into **numbers**.

## A vector is just a list of numbers

When we line numbers up, we call it a **vector**. A movie might become `[4.5, 0, 1, 120]` (rating, is-horror, is-sequel, minutes). A word becomes a long vector of numbers that captures its meaning. Same idea, every time.

```python
movie = [4.5, 0, 1, 120]
```

Each slot is a **feature** — one measurable aspect of the thing. Stack many examples and you have a dataset: a list of vectors.

<Callout type="note">
"Turn it into numbers, then do math on the numbers." That sentence describes nearly all of machine learning.
</Callout>

## Summarizing a vector: the average

The simplest thing you can do with a vector is collapse it to one number. The **average** (mean) shows up everywhere — normalizing data, computing loss, reporting results.

<CodeExercise

  starter={`ratings = [4.5, 3.0, 5.0, 4.0]

`}
  expected="4.12"
  solution={`ratings = [4.5, 3.0, 5.0, 4.0]

## Comparing two vectors: the dot product

Here's the operation that powers search, recommendations, and attention inside transformers: the **dot product**. Multiply two vectors slot-by-slot and add it all up. The bigger the result, the more the two vectors "point the same way" — i.e., the more similar they are.

```text
a · b = a1*b1 + a2*b2 + a3*b3 + ...
```

<CodeExercise
  id="vec-dot"
  prompt="Print the dot product of a and b (1*4 + 2*5 + 3*6)."
  starter={`a = [1, 2, 3]
b = [4, 5, 6]
# Print the dot product of a and b
`}
  expected="32"
  solution={`a = [1, 2, 3]
b = [4, 5, 6]
print(sum(x * y for x, y in zip(a, b)))`}
/>

## How long is a vector?

A vector also has a **length** (magnitude): the square root of the sum of its squares. For `[3, 4]` the sum of squares is `9 + 16 = 25`, so the length is `5`. Length is how we *normalize* vectors so we compare direction, not size.

<CodeExercise
  id="vec-sqsum"
  prompt="Print the sum of squares of v (the part inside the square root of its length)."
  starter={`v = [3, 4]
# Print the sum of each element squared
`}
  expected="25"
  solution={`v = [3, 4]
print(sum(x * x for x in v))`}
/>
