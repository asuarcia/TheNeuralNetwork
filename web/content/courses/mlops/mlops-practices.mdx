You can train, serve, and monitor. The last MLOps skill is **changing things safely** — because in production, a new model is a risk until proven otherwise. Borrow the hard-won practices of software engineering and adapt them to ML.

## Version everything

In normal software you version code. In ML you must version **three** things, because any of them changes the result:

- **Code** — the training and serving logic (Git).
- **Data** — which dataset and version trained this model.
- **Model** — the resulting weights, stored in a **model registry** with its metrics and lineage.

If you can't say exactly which code + data produced a given model, you can't reproduce, debug, audit, or safely roll back. Reproducibility is non-negotiable.

## CI/CD for models

**Continuous integration / continuous delivery** for ML means automating the path from change to production with checks at each gate: run the test suite, retrain, **evaluate against the current production model**, and only deploy if it passes. The evaluation set is your safety net — nothing ships that fails it (exactly the discipline from the AI Engineering eval lesson).

## Roll out gradually: canary deploys

Never flip 100% of traffic to a new model at once. A **canary** (or A/B) rollout sends a *small slice* of traffic to the new version, watches its metrics, and only then ramps up. If something's wrong, the blast radius is tiny.

While the canary runs, you compare its error rate to the current model and **roll back automatically** if it's worse.

<CodeExercise
  id="ops-rollback"
  prompt="The canary's error rate is 0.08; the baseline's is 0.03. Print 'rollback' if the canary is worse, else 'keep'."
  starter={`canary_error = 0.08
baseline_error = 0.03
# Print "rollback" if canary is worse than baseline, else "keep"
`}
  expected="rollback"
  solution={`canary_error = 0.08
baseline_error = 0.03
print("rollback" if canary_error > baseline_error else "keep")`}
/>

The canary failed, so it's pulled before most users ever saw it. That's the whole point: **fail small.**
