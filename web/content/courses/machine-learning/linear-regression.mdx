**Linear regression** fits a straight line through your data and uses it to predict. It's the "hello world" of machine learning — and the core idea inside far more complex models.

## The model is one equation

```text
prediction = w * x + b
```

- `w` (**weight**) is the slope: how much the prediction rises per unit of input.
- `b` (**bias**) is the intercept: the prediction when the input is zero.

If price rises by 3 (thousand) per square meter, then `w = 3`. A 100&nbsp;m² house predicts `3 * 100 = 300`.

```python
def predict(x, w, b):
    return w * x + b

print(predict(100, 3, 0))   # 300
```

**Learning** linear regression means finding the `w` and `b` that make the line fit the data best — usually by minimizing error, which is the next lesson.
