**Linear regression** fits a straight line through your data and uses it to predict. It's the "hello world" of machine learning — and the core idea inside far more complex models, including the neuron you'll build in the next track.

## The model is one equation

```text
prediction = w * x + b
```

- `w` (**weight**) is the slope: how much the prediction rises per unit of input.
- `b` (**bias**) is the intercept: the prediction when the input is zero.

If price rises by 3 (thousand) per square meter, then `w = 3`. A 100&nbsp;m² house predicts `3 * 100 = 300`.

```python
def predict(x, w, b):
    return w * x + b

print(predict(100, 3, 0))   # 300
```

## What does "best fit" mean?

There are infinitely many lines. **Learning** means picking the `w` and `b` whose line sits closest to the data. To make "closest" precise, we measure error with **Mean Squared Error (MSE)** — the average of the squared gaps between prediction and truth:

```text
MSE = average of (prediction - actual)²
```

Squaring does two things: it makes all errors positive, and it punishes big misses far more than small ones. The best line is the one with the **smallest MSE**.

<CodeExercise
  id="lr-mse"
  prompt="The data follows y = 2x exactly, and w=2, b=0. Print the MSE (it should be 0.0 — a perfect fit)."
  starter={`xs = [1, 2, 3]
ys = [2, 4, 6]   # true relationship: y = 2x
w, b = 2, 0
# Print the mean squared error of w*x + b vs ys
`}
  expected="0.0"
  solution={`xs = [1, 2, 3]
ys = [2, 4, 6]
w, b = 2, 0
preds = [w * x + b for x in xs]
print(sum((p - y) ** 2 for p, y in zip(preds, ys)) / len(ys))`}
/>

Zero error — the line passes through every point. Change `w` to 3 and watch the MSE shoot up. Training is the search for the `w` and `b` that drive this number to its minimum, which is exactly what gradient descent does later in this track.

## More than one feature

Real problems have many features — size, bedrooms, age. The model just gains one weight per feature, and the prediction becomes a **dot product** plus the bias (you built dot products in the Python for AI track):

```text
prediction = w₁x₁ + w₂x₂ + ... + b
```

<Callout type="tip">
That weighted sum plus a bias — <code>w·x + b</code> — is <em>literally</em> what a single neuron computes. Linear regression isn't just a model; it's the atom that deep learning is built from. Learn it cold here and the next track is half done.
</Callout>
