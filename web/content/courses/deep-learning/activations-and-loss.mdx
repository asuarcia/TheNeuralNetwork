A neuron's weighted sum `z` is linear. Stack a thousand linear steps and you still get... a line. To model curves, language, and images, networks need **non-linearity** — and that comes from the **activation function**.

## ReLU: the workhorse

The most common activation is almost insultingly simple — **ReLU** (Rectified Linear Unit): keep positives, zero out negatives.

```text
ReLU(z) = max(0, z)
```

```python
def relu(z):
    return max(0.0, z)

print(relu(3.0))    # 3.0
print(relu(-2.0))   # 0.0
```

That tiny "bend" at zero is enough: stack ReLUs across layers and the network can approximate almost any function.

## Loss: the thing we minimize

To learn, the network needs a score for how wrong it is — the **loss**. You already met one: Mean Absolute Error. Training is just nudging weights to push the loss down.

<Callout type="tip">
Activation gives the network <strong>expressiveness</strong> (it can bend). Loss gives it a <strong>goal</strong> (get the number down). Next lesson connects them: how to nudge weights to reduce loss.
</Callout>
