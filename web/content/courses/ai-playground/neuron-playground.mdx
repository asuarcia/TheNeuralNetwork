A neural network is millions of copies of one tiny operation. Instead of reading about it, let's *play* with it until it's obvious. A neuron takes inputs, multiplies each by a **weight**, adds a **bias**, and squashes the result with an **activation** (here, ReLU = keep positives, zero out negatives).

<PredictThenReveal prompt="If you set both weights to 0, what will the neuron output — no matter what the inputs are?">
Just the bias, passed through ReLU. With weights at 0 the inputs can't matter, so `z = bias`, and the output is `max(0, bias)`. Try it below: set w₁ and w₂ to 0 and drag the inputs — nothing changes.
</PredictThenReveal>

## Your live neuron

Drag everything. Watch `z` (the weighted sum) and the ReLU output update instantly. Two goals: make the neuron **fire hard**, then make it go completely **silent**.

Notice the pattern: the output only responds to an input when that input has a non-zero weight. The weights are what the network *learns* — they decide which inputs matter and how much.
