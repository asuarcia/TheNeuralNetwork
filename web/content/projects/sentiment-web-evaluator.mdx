## 2. Classify a piece of text

Count positive minus negative words; the sign is the sentiment. Build it and test on a clearly positive review.

<CodeExercise
  id="se-classify"
  prompt="Write sentiment(text) returning 'positive', 'negative', or 'neutral', and test it."
  starter={`positive = {"love", "great", "amazing", "good", "happy", "best"}
negative = {"hate", "bad", "terrible", "awful", "sad", "worst"}

def sentiment(text):
    words = text.lower().split()
    score = sum(w in positive for w in words) - sum(w in negative for w in words)
    return "positive" if score > 0 else "negative" if score < 0 else "neutral"

## 4. Upgrade to an LLM classifier

The lexicon misses sarcasm, negation ("not bad"), and nuance. The modern approach: send each text to an LLM with a prompt like *"Classify the sentiment as positive/negative/neutral. Return only the label."* — using the model as a structured-output engine.
