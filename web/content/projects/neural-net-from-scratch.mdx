## 2. The forward pass

We'll use a 2 → 4 → 1 network (two inputs, a hidden layer of four neurons, one output), with a sigmoid squash. Initialize the weights and push the data through once.

## 3. Train it with backpropagation

Now the full loop: forward pass, then backprop the error through both layers, then a gradient step — 5000 times. By the end the network should output ≈ [0, 1, 1, 0].

<CodeExercise
  id="nn-train"
  prompt="Train the network on XOR and print the rounded predictions (expect 0, 1, 1, 0)."
  starter={`import numpy as np
np.random.seed(0)
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([[0], [1], [1], [0]])
def sigmoid(z):
    return 1 / (1 + np.exp(-z))

W1 = np.random.randn(2, 4); b1 = np.zeros((1, 4))
W2 = np.random.randn(4, 1); b2 = np.zeros((1, 1))
lr = 0.5

for i in range(5000):
    h = sigmoid(X @ W1 + b1)            # forward
    out = sigmoid(h @ W2 + b2)
    d_out = (out - y) * out * (1 - out)  # backprop
    d_h = (d_out @ W2.T) * h * (1 - h)
    W2 -= lr * h.T @ d_out               # update
    b2 -= lr * d_out.sum(0, keepdims=True)
    W1 -= lr * X.T @ d_h
    b1 -= lr * d_h.sum(0, keepdims=True)

print(np.round(out).ravel())`}
  solution={`import numpy as np
np.random.seed(0)
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([[0], [1], [1], [0]])
def sigmoid(z):
    return 1 / (1 + np.exp(-z))
W1 = np.random.randn(2, 4); b1 = np.zeros((1, 4))
W2 = np.random.randn(4, 1); b2 = np.zeros((1, 1))
lr = 0.5
for i in range(5000):
    h = sigmoid(X @ W1 + b1)
    out = sigmoid(h @ W2 + b2)
    d_out = (out - y) * out * (1 - out)
    d_h = (d_out @ W2.T) * h * (1 - h)
    W2 -= lr * h.T @ d_out
    b2 -= lr * d_out.sum(0, keepdims=True)
    W1 -= lr * X.T @ d_h
    b1 -= lr * d_h.sum(0, keepdims=True)
print(np.round(out).ravel())`}
/>

It learned XOR — a problem impossible without the hidden layer. You just implemented forward propagation, backpropagation, and gradient descent by hand. This *is* deep learning; frameworks like PyTorch are this, vectorized and on a GPU.
