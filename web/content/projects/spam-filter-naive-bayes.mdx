## 3. Score a new message

To classify, we score a message under each class by multiplying its word probabilities. We add **+1 smoothing** so an unseen word doesn't zero out the whole score. Whichever class scores higher wins.

<CodeExercise
  id="nb-classify"
  prompt="Score 'win free money' under each class and print the predicted label."
  starter={`from collections import Counter
data = [
    ("win money now", "spam"), ("win a free prize", "spam"), ("cheap meds win", "spam"),
    ("meeting at noon", "ham"), ("lunch tomorrow", "ham"), ("project update meeting", "ham"),
]
spam_words, ham_words = Counter(), Counter()
for text, label in data:
    (spam_words if label == "spam" else ham_words).update(text.split())

def score(words, counts):
    total = sum(counts.values())
    vocab = len(set(spam_words) | set(ham_words))
    s = 1.0
    for w in words:
        s *= (counts[w] + 1) / (total + vocab)   # +1 smoothing
    return s

msg = "win free money".split()
label = "spam" if score(msg, spam_words) > score(msg, ham_words) else "ham"
print(label)`}
  expected="spam"
  solution={`from collections import Counter
data = [
    ("win money now", "spam"), ("win a free prize", "spam"), ("cheap meds win", "spam"),
    ("meeting at noon", "ham"), ("lunch tomorrow", "ham"), ("project update meeting", "ham"),
]
spam_words, ham_words = Counter(), Counter()
for text, label in data:
    (spam_words if label == "spam" else ham_words).update(text.split())

    total = sum(counts.values())

    s = 1.0

    return s
msg = "win free money".split()
label = "spam" if score(msg, spam_words) > score(msg, ham_words) else "ham"
print(label)`}
